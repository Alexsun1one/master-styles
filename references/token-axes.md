# Token Axes

These are the reusable visual dimensions. Every master in `master-library.md` is described
along them. When you plan an image, you are picking values on these axes — not copying a
picture. Talk in axes, not artist names.

- **Palette logic** — the RULE governing color, not the colors. E.g. "two flat clashing
  colors + black ground", "near-white + one ink tone", "primaries rationed into a white
  field", "one hue family glowing from within". The logic transfers even when you change the
  actual colors.

- **Composition / structure** — how the frame is organized: mathematical grid, asymmetric
  balance, dead-center frontality, diagonal/radial thrust, rule-of-thirds, dramatic scale
  contrast, corner-compression, horizon placement.

- **Shape & line** — the vocabulary of form: geometric primitives, organic scissor-cut
  shapes, bold contour key-line, hairline, flat vector, cut-paper jagged edges, feathered/no
  edge.

- **Negative space / density** — generous emptiness (active void / *ma*) vs. controlled
  balance vs. maximal fill. One of the most powerful selectors; often the whole identity.

- **Light & shadow** — flat no-shadow, hard graphic shadow, soft directional falloff, raking
  long shadows, internal glow, butterfly/under-nose, cinematic chiaroscuro.

- **Texture & material** — flat litho print, matte gouache, screenprint, gold leaf + relief,
  woodblock grain, platinum-print tonal scale, real photographed objects.

- **Motif / signature device** — the recognizable trick: dual-meaning negative space,
  type-as-image, one-symbol reduction, faceless figures, rising-sun rays. Borrow the device,
  invent your own instance.

- **Typography signature** (designers) — typeface character, scale, placement, integration
  (caption beside vs. built into the image).

- **Restraint principle** — the ONE thing the master REFUSES to do. This is what makes the
  work elegant instead of busy. Skipping the restraint is the #1 reason AI output looks
  generic — the refusal is the point.

- **Transfer note** — how to apply the tokens to a DIFFERENT subject without copying actual
  works. The anti-pastiche guard.

## How to use the axes

1. Pick the **feeling** you need (elegance, punch, warmth, gravitas, narrative, opulence, loud).
2. Choose a **structure** value from one master and a **palette/light** value from another.
3. Lock one shared **restraint** so the two sources don't fight.
4. Write the prompt in axis terms applied to the user's subject — never "in the style of X".
