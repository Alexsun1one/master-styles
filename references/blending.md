# Choosing & Blending

Pick by the TOKEN you need, not by the famous name. Then combine at most two sources with one
shared restraint. This is what produces a high-end, art-directed look instead of pastiche.

## Choose by feeling → token source

- **Elegance via emptiness** → Kenya Hara (void / *ma*), Irving Penn (neutral seamless + soft
  side light), Malika Favre (negative space carving the subject), Rothko (glowing calm fields).
- **Bold graphic punch** → Saul Bass (one-symbol cut-paper), Cassandre (heroic low-angle +
  caps), Müller-Brockmann (grid + one accent), Noma Bar (dual-image), Hokusai (key-line +
  scale contrast).
- **Warmth / joy / approachability** → Olimpia Zagnoli (rounded saturated pop), Hockney (sunny
  flat planes), Matisse cut-outs (pure-color rhythm).
- **Gravitas / authority on a face** → Platon (face-on-void), Penn (contained corner),
  Müller-Brockmann / Vignelli (institutional grid).
- **Narrative / cinematic mood** → Annie Leibovitz (staged saturated world), Tom Haugomat
  (faceless figure + two-tone light), Hopper (raking light + solitude).
- **Premium / opulent / sacred** → Klimt (gold ground + ornament + one realistic anchor).
- **Loud cultural energy** → Paula Scher (type-as-image), Tadanori Yokoo (maximalist collage +
  rising-sun rays).
- **Calm modern restraint** → Geoff McFetridge (muted matte faceless), Hara, Vignelli.

## The blend recipe

Take the **composition/structure** from one master and the **palette or light** from another,
then keep ONE shared restraint so they don't fight.

- Swiss grid (structure: Müller-Brockmann) + Matisse cut-out palette (pure flats on white) →
  disciplined yet joyful poster.
- Hopper raking light (light) + Hara emptiness (negative space) → quiet, cinematic, breathing
  cover.
- Platon face-on-void (crop + background) + Rothko glow (palette as a colored void instead of
  pure white) → a monumental portrait with emotional color.
- Noma Bar dual-image (motif) + Bass hot-color-on-flat (palette) → a witty single-symbol cover.
- Ikko Tanaka geometric reduction (shape) + Zagnoli saturated polychrome (palette) → a modern,
  friendly, iconic mark.

## The rule of thumb

> One structure source + one color/light source + one shared restraint.

Three token sources is the ceiling; two is usually best. More than that turns to mud.

## Picking the shared restraint

Both sources must obey the same refusal, or they fight. Examples:

- "no gradients, all flat" (works for Matisse + Müller-Brockmann)
- "no environment, subject on a void" (works for Platon + Rothko)
- "no decoration, every position justified" (works for Vignelli + Hara)
- "no facial detail, emotion via scale and light" (works for Haugomat + McFetridge)

The restraint is not optional. A master's elegance is the refusal. Keep it explicit in the
prompt.
