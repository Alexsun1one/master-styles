## 同一主题多视觉语言

```text
用 master-styles 给“同一杯春茶”做一张 16:9 风格谱系图。

要求：
- 同一主题分成四格：墨色、几何、光影、花版
- 每格都画一杯春茶，但材质、构图、线条和色彩明显不同
- 不要写艺术家名字，不要模仿具体名作
- 输出 style spec，并写出每格的 token：材质 / 构图 / 色彩 / 线条 / 克制原则
```
